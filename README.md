# ticketfinder 티켓 예매 사이트
Spring MVC Web Project


# 설명
팀원 : 이규민, 김희연, 김태형, 임라경, 김지선

현재 운영되고 있는 사이트를 모방하여 만든 티켓 예매 사이트.

Spring Framework 와 다양한 기술을 활용하여 웹 사이트 구축.
